## üí° Key Insight

> Code Riders used an LLM to generate "smarter" responses, but **the GUVI rubric doesn't score response quality** ‚Äî it scores:
> 1. Was scam detected? (boolean)
> 2. Was intelligence extracted? (phone, bank, UPI, links, email)
> 3. Was engagement sustained? (message count, duration)
> 4. Was JSON structure correct? (field names, types)
>
> **None of these require an LLM.** By removing the LLM, WebCheers eliminated the #1 source of latency, cost, and failure ‚Äî while scoring higher.
